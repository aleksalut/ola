Database Tables

AspNetUsers
Identity user accounts + custom profile fields
Habits
User-defined habits to track daily
DailyProgresses
Daily progress entries for habits
Goals
User goals with priorities and deadlines
EmotionEntries
Emotion journal with 5-point scales
AuditLogs
Activity audit trail for all actions
AspNetRoles
System roles (Admin, User)
AspNetUserRoles
User-role mapping (junction table)

?? Primary Keys

  • AspNetUsers.Id: varchar(450)
  • Habits.Id: int (auto-increment)
  • DailyProgresses.Id: int
  • Goals.Id: int
  • EmotionEntries.Id: int
  • AuditLogs.Id: int

Entity Relationships

User ? Habits (1:N)

FK: Habits.UserId ? AspNetUsers.Id

Delete: RESTRICT

Index: IX_Habits_UserId_Name

User ? Goals (1:N)

FK: Goals.UserId ? AspNetUsers.Id

Delete: RESTRICT

Index: IX_Goals_UserId_Status_Priority

User ? EmotionEntries (1:N)

FK: EmotionEntries.UserId ? AspNetUsers.Id

Delete: RESTRICT

Index: IX_EmotionEntries_UserId_Date

Habit ? DailyProgresses (1:N)

FK: DailyProgresses.HabitId ? Habits.Id

Delete: CASCADE

Note: Progress deleted with habit

User ? DailyProgresses (1:N)

FK: DailyProgresses.UserId ? AspNetUsers.Id

Delete: RESTRICT

Index: IX_DailyProgresses_UserId_HabitId_Date

User ? AuditLogs (1:N)

FK: AuditLogs.UserId ? AspNetUsers.Id

Delete: RESTRICT

Index: IX_AuditLogs_UserId_Timestamp
