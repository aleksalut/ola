?? How to Access Reports

  1. Resume Backend: In Visual Studio, press F5 or click Continue ??
  2. Open Application: Click "Open Frontend" button above
  3. Login: Use your credentials (e.g., demo@example.com / Demo1234!)
  4. Navigate: Click "Reports" in the top navigation bar
  5. Explore: View your statistics, charts, and insights!

? Dashboard Features

  • 8 Statistics Cards - Habits, Goals, Completion Rate, Progress, etc.
  • Habit Progress Chart - Interactive line chart with habit selector
  • Emotion Trends Chart - Multi-line chart tracking 5 emotions over 30 days
  • Quick Insights - Success rate, active items, journal entries
  • Responsive Design - Works on mobile, tablet, and desktop

?? Troubleshooting

  • 401 Unauthorized: You need to login first
  • No Data: Create some habits/goals/emotions first
  • Backend Connection Failed: Resume debugger in Visual Studio
  • Charts Not Loading: Check browser console (F12) for errors

?? API Endpoints

  • GET /api/reports/my-statistics - User statistics
  • GET /api/reports/completion-rate - Goal completion %
  • GET /api/reports/habit-progress/{id} - Habit progress history
  • GET /api/reports/emotion-trends?days=30 - Emotion trends
